# Smart Online Book Store

A fully responsive and interactive online bookstore web application built using HTML, CSS, and JavaScript. This project provides a real-world e-commerce experience where users can browse books, search by title or author, filter by category, manage a shopping cart, and maintain a wishlist.

## Features

* Browse a wide collection of books
* Search books by title or author
* Filter books by category
* Add books to cart
* Update cart quantity
* Remove books from cart
* Add books to wishlist
* Featured books dashboard
* Real-time cart and wishlist updates
* Data persistence using Local Storage
* Fully responsive design for desktop, tablet, and mobile devices
* Interactive event handling for a seamless user experience

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage API

## Key Concepts Implemented

* DOM Manipulation
* Event Handling
* Array Methods (`filter`, `map`, `find`, `forEach`)
* Responsive Web Design
* Dynamic Content Rendering
* Client-Side Data Storage

## Project Structure

* `index.html` – Main structure of the application
* `style.css` – Styling and responsive layout
* `script.js` – Application logic and interactivity

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Run `index.html` in any modern web browser.
